# Disaster-Management-in-Himachal-Pradesh-
A Python-based system using 4 collaborating ML models to predict disasters and optimize relief operations. CNN (PyTorch) ,LSTM (PyTorch) — Forecasts 7-day rainfall from 30-day history,Random Forest — Scores terrain/elevation features,XGBoost — Scores historical pattern features,Ensemble Predictor- Weighted fusion of all 4 models
